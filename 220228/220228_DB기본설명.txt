DataBase(자료집합) = DB의 3대 기능(정의기능, 조작기능, 제어기능)

물리적DB => 저장된DB
논리적DB => DBMS(정의어, 조작어, 제어어)=>데이터베이스관리자(DBA)
조작(기능)어[SQL] => 선택(select), 추가(insert), 변경(update), 삭제(delete)
DB사용자 => DBA(관리자=DB정의, 조작, 제어)
일반사용자 => SQL을 사용하여 선택, 추가, 변경, 삭제
응용프로그래머 => 외부(응용)프로그램 내부에 SQL을 사용

DB 3대 기능
1.정의(어) 기능
	1.1 DB 생성, table 생성 (create)

2. 조작(어) 기능
	2.1 검색(select)
	2.2 삽입(insert)
